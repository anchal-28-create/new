# Pong Game – Python (Pygame)

Pong Game developed using Python and Pygame, featuring custom graphics, sounds, player name input, scoring system, and replay functionality.
The game provides an interactive arcade-style experience with smooth controls and visual effects.

1.Project Objective
Create a classic Pong game using Python
Implement real-time player controls
Add sound effects and graphics
Track scores and declare a winner
Provide a replay option for continuous gameplay

2.Features
 Two-player gameplay
 Player name input at the start
 Paddle and ball collision detection
 Sound effects on paddle hit
 Real-time score display
 Winner announcement
Play-again option
 Custom background, paddle, and ball images
Custom font support

3.Technologies Used
Python 
Pygame – Game development
Custom assets (images, sound, font)
